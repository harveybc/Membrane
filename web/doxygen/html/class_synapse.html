Detailed Description

Models a synapse from the axon of an integrate and fire spiking Neuron. The image bellow shows examples of some types of synapses.
.

Image source: Blausen.com staff. "Blausen gallery 2014". Wikiversity Journal of Medicine. DOI:10.15347/wjm/2014.010. ISSN 20018762. - Own work
Warning
The image's synapse type numbering does not correspond with the actual numbering of all the types of synapes used.

Synapses are the inputs of neurons and receive data from the axon of a Neuron. This class models the connection to a source Neuron::axon.

Epigenetics

Epigenetics are modeled using the Synapse::prob_inheritable attribute wich increases or decreases due to external factors (the ANN fitness).

Relationships with other classes
The Synapse class is the base class of PlasticSynapse used in the vectors of synapses by type for each Neuron, all the types of synapses are implemented in the SpikingEvaluator class wich evauates all Neurons in a NeuralNetwork.
