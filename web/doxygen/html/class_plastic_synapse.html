Detailed Description

Chemical synapse with long/short-term plasticity.

Synaptic plasticity modifies the strength of a synapse temporarily depending on the activity of the neuron.

Types of synaptic plasticity
There are two types of synaptic plasticity depending on their mechanics and the time required to return the Synapse to it´s baseline strength:
-Short-term plasticity (STP): acts as a band-pass filter of the pre-synaptic activity, is added to the base line strength of the synapse.
-Long-term plasticity (LTP): uses hebbian learning and it´s intensity depends on the phase shift of the post-synaptic activity respect to the pre-synaptic activity, is added to the baselinestrengh of the synapse.
Epigenetics
Epigenetics are modelled as inheritable changes in the strength of a connection related to external factors (fitness of the Neural Network).
Relationships with other classes
The Synapse class is the base class of PlasticSynapse used in the vectors of synapses by type for each Neuron, all the types of synapses are implemented in the SpikingEvaluator class wich evauates all Neurons in a NeuralNetwork.
